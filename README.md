# helmets-kenya

<img src="https://nasaharvest.github.io/assets/helmets-cover.png" />

Code for paper: "Helmets Labeling Crops: Kenya Crop Type Dataset Created via Helmet-Mounted Cameras and Deep Learning".

### Repository Structure

`notebooks/` contains code for transforming GoPro photos into a crop type dataset.

`gee/` contains code for visualizing and creating a maize map using the crop type dataset.
